# Installation

## From the WordPress.org plugin directory

1. In your WordPress admin, go to **Plugins → Add New**.
2. Search for **Museum for WordPress**.
3. Click **Install Now**, then **Activate**.

## From a release zip

1. Download the latest release from the [GitHub releases page](https://github.com/mikethicke/wp-museum/releases/latest).
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Choose the zip file and click **Install Now**, then **Activate**.

Alternatively, extract the zip into `wp-content/plugins/` and activate from the **Plugins** screen.

## First steps

After activation:

1. Visit **Museum → Object Kinds** to define your first object type and its custom fields. See [Object Kinds & Fields](object-kinds.md).
2. Create one or more collections via **Collections**. See [Collections](collections.md).
3. Start cataloging objects from the custom post type sidebar.
4. Use the [Museum blocks](blocks.md) in the block editor to display objects and collections anywhere on your site.
