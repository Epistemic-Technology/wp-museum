# Museum for WordPress

A full-featured museum management system built directly into WordPress. Catalog museum objects with custom fields and image galleries, organize them into hierarchical collections, and embed them anywhere on your site with Gutenberg blocks.

## Features

- Define custom **object kinds** (e.g. artifacts, specimens, instruments) with per-kind custom fields.
- Per-object **image galleries** with a primary image and unlimited additional images.
- **Collections** with hierarchical parent/child relationships and a dedicated custom taxonomy.
- A rich set of **Gutenberg blocks** for search, display, and navigation.
- **REST API** (`wp-museum/v1`) for objects, collections, kinds, fields, and image attachments.
- **Museum Remote** companion plugin for embedding collections from a central museum site on other WordPress sites.
- **Capability-based security** — custom roles and capabilities control access independently of standard post permissions.
- Fully internationalizable (`wp-museum` text domain).

## Requirements

- WordPress 6.9 or later
- PHP 8.2 or later
